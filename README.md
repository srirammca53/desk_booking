# desk_booking

1. Clone the application in to local environment using 
  
   git clone https://github.com/srirammca53/desk_booking.git

2. next move to the app root folder 
  
   cd desk_booking

3. data base creation and migrate

   rake db:create
   rake db:migrate
   rake db:seed

4. run the rails server
   
   rails s

5. Versions
 
   Ruby 2.2.0
   Rails 4.2.10
