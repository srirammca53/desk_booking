# desk_booking

* Clone the application in to local environment using 
  
   git clone https://github.com/srirammca53/desk_booking.git

*  next move to the app root folder 
  
   cd desk_booking

* data base creation and migrate

   rake db:create
   rake db:migrate
   rake db:seed

* run the rails server
   
   rails s

*  Versions
 
   Ruby 2.2.0
   Rails 4.2.10


*  Steps to follow
	  1. rake db:create
	  2. rake db:migrate
	  3. rake db:seed
	  4. rails server
